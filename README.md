### 🎮 플레이 방법

이동: WASD

점프: Space bar

상호작용: E
- 몬스터 공격
- 물 마시기
- 자원 줍기

음식 먹기: 인벤토리 우클릭

퀵 슬롯 등록: 드래그 앤 드랍
퀵 슬롯 사용: 1 2 3 4 5

건축물
- 청사진 생성: 마우스 좌클릭
- 청사진 회전: Z, C
- 청사진 취소: Esc
- 건축물 생성: 청사진 생성 후 마우스 좌클릭
- 건축물 철거모드 및 철거모드 취소: B
